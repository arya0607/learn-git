## Useful notation

`@` is a synonym for `HEAD`, the currently checked-out commit.

If you have a refspec, you can append `~` to it to mean "the commit before this
one."

	3455c89 = HEAD = @
	b14c16d = HEAD~ = @~
	13ed675 = HEAD~~ = @~~ = master
	017986c = HEAD~~~ = @~~~ = master~
	5516fd0 = HEAD~~~~ = @~~~~ = master~~

## `git show`

You can see what a commit did with `git show`.

	$ git show @~
